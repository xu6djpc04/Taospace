# 使用者文案排版與用語習慣紀錄

> **最後更新**：115-06-14
> **用途**：紀錄使用者處理「活動發布文案」、「會議預報文案」時的個人排版偏好與慣用語。

## 1. 排版習慣

- **標題與項目不用粗體**：請維持純文字呈現，不使用 Markdown 的 `**` 粗體語法（例如：直接寫 `一、時間：`，不要寫 `**一、時間：**`）。
- **增加段落間距**：各段落、各流程項目之間，請多空一行（Double spacing），保持畫面乾淨。

## 2. 用語習慣

- 使用 **「接待」** 代替「招待」。
- 使用 **「專題：」** 代替「專題分享：」。
- 單位名稱請**展開**：遇到縮寫時盡量展開，例如「一二中」請寫為「一中、二中」。

## 3. 資訊簡化（化繁為簡）

- **分組討論**：只需列出 `分組討論：` 與時間即可，**不需**列出各組討論的細項題目。
- **工作分配**：只需寫出負責的單位或組別即可（如：`班務組`），**不需**特別加註人名（如不需寫成 `班務組 (穎締)`）。
